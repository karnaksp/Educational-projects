Quickstart
----------
To get started with the Voight-Kampff Test:

1. **Use Makefile to simulate user behavior:**
   ```make Ex00```

**Or check you on replicant or human detected:**
   ```make create_env && cd Ex00 && python main.py```

2. **Use Makefile to check testing coverage:**
   ```make Ex01```

3. **Finally, to create documentation, use:**
   ```make html```
